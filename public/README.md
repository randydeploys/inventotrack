# Assets Vuexy

Ce dossier contient les assets du thème Vuexy (payant).

## Installation

1. Acheter Vuexy sur ThemeForest (si pas déjà fait)
2. Télécharger le package Vuexy
3. Extraire le dossier `assets/` ici (`public/assets/`)

## Structure attendue
```
public/assets/
├── css/
├── js/
├── vendor/
├── img/
└── ...
```
